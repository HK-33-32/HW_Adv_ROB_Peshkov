## О репозитории

Это репозиторий с реализациями домашних заданий по курсу **Advanced Robotics**. Здесь собраны пять практических работ, охватывающих ключевые темы современной робототехники: фильтрацию данных, детекцию линий и плоскостей, оценку состояния (Kalman-фильтры и EKF), 3D-реконструкцию и планирование пути.

## Структура репозитория и домашние задания

| # | Название домашнего задания | Краткое описание | Ссылка на папку с реализацией |
|---|----------------------------|------------------|--------------------------------|
| 1 | **HW1: Wall Detection (Hough Transform) + Ground Detection (RANSAC)** | 1.1 — Поиск прямых линий (стен) в 2D-облаке точек LIDAR с помощью преобразования Хафа. 1.2 — Выделение плоскости пола из 3D-точек с помощью RANSAC. Данные: Intel Research Lab + KITTI. Результат: отрисовка линий поверх точек и цветовая сегментация пола/препятствий. | [HW1-Line-Plane-Detection](HW1-Line-Plane-Detection) |
| 2 | **HW2: Linear Kalman Filter for Smartphone sensors** | Линейный Kalman-фильтр для оценки состояния по данным акселерометра и гироскопа смартфона. Реализация предсказания и коррекции, визуализация траекторий и ошибок. | [HW2_Linear_Kalman_Filter](HW2_Linear_Kalman_Filter) |
| 3 | **HW3: Euler-based vs Quaternion-based EKF for Smartphone sensors** | Сравнительный анализ Extended Kalman Filter на углах Эйлера (вариант A) и кватернионах (вариант B). Реальные данные акселерометра + гироскопа, графики Roll/Pitch/Yaw, анализ нормы кватерниона и вычислительной ошибки. | [HW3_EKF_Euler_vs_Quaternion](HW3_EKF_Euler_vs_Quaternion) |
| 4 | **HW4: Gaussian Splatting for 3D Reconstruction** | 3D-реконструкция реального объекта методом Gaussian Splatting (с SfM/COLMAP). Сравнение с классическим Structure-from-Motion. Эксперименты с обучением, анализ влияния количества итераций, визуализация сплатов и финального рендера. | [HW4_Gaussian_Splatting](HW4_Gaussian_Splatting) |
| 5 | **HW5: Path Planning (A vs RRT/RRT)** | Планирование пути в 2D-окружении: классический A* vs RRT/RRT* + Path Smoothing. Сравнение по времени, количеству узлов, длине пути и качеству траектории в узких коридорах. Визуализация карт и найденных путей. | [HW5_Path_Planning](HW5_Path_Planning) |

- Один или несколько `.ipynb`-файлов.
- Результаты (изображения, GIF/MP4 видео траекторий/рендеров, таблицы).
